# app/ — FastAPI Application

The containerised Python application that runs on the EKS cluster. Demonstrates Karpenter autoscaling by being the workload that triggers node provisioning.

## `main.py` — Line-by-Line

```python
from fastapi import FastAPI
import os
# os.getenv() reads environment variables injected by Kubernetes
# (NODE_NAME, POD_NAME from fieldRef; GOOGLE_API_KEY from secretKeyRef)

app = FastAPI(title="Hello World API", version="1.0.0")
# Creates the FastAPI application instance.
# title and version appear in the auto-generated OpenAPI docs at /docs.

@app.get("/")
def root():
    return {
        "message": "Hello from FastAPI on EKS with Karpenter!",
        "version": "1.0.0",
        "node": os.getenv("NODE_NAME", "unknown"),
        # NODE_NAME = the EC2 instance hostname this pod is running on.
        # Set via deployment.yaml env.fieldRef.spec.nodeName
        # Change between requests = load balancer sent you to a different pod/node.
        "pod": os.getenv("POD_NAME", "unknown"),
        # POD_NAME = the Kubernetes pod name.
        # Set via deployment.yaml env.fieldRef.metadata.name
    }

@app.get("/health")
def health():
    return {"status": "healthy"}
    # Used by readinessProbe and livenessProbe in deployment.yaml.
    # Kubernetes sends HTTP GET /health every few seconds.
    # If this returns non-2xx or times out, the pod is removed from Service endpoints.
```

**Endpoints:**

| Method | Path | Description |
|---|---|---|
| GET | `/` | Returns hello message + pod/node info |
| GET | `/health` | Returns `{"status": "healthy"}` — used by K8s probes |
| GET | `/docs` | Swagger UI (auto-generated by FastAPI) |
| GET | `/openapi.json` | OpenAPI spec |

## `Dockerfile` — Line-by-Line

```dockerfile
FROM python:3.12-slim
# Minimal Python 3.12 base image (~150 MB vs ~1 GB for full image).
# "slim" excludes build tools, documentation, and dev packages.

WORKDIR /app
# All subsequent commands run in /app. Files are copied here.

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
# Install dependencies BEFORE copying app code.
# Docker caches each layer. If only main.py changes, pip install is NOT re-run.
# --no-cache-dir: don't store the pip download cache in the image (smaller image).

COPY main.py .
# Copy app code after dependencies. This layer is invalidated on every code change,
# but it's cheap because there's nothing to install here.

RUN adduser --disabled-password --gecos "" appuser
USER appuser
# Run as non-root user (security best practice).
# Prevents the container from having root access to the host if escaped.
# --disabled-password: no password login
# --gecos "": no GECOS info (name, phone, etc.)

EXPOSE 8000
# Documents that the container listens on port 8000.
# Does NOT actually open a port — that's done by the Service in K8s.

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
# Start the ASGI server.
# main:app       — module "main", FastAPI instance "app"
# --host 0.0.0.0 — listen on all interfaces (not just loopback)
# --port 8000    — matches containerPort in deployment.yaml
```

## Building and Pushing the Image

```bash
# Authenticate to ECR
aws ecr get-login-password --region us-east-1 | \
  docker login --username AWS --password-stdin ACCOUNT_ID.dkr.ecr.us-east-1.amazonaws.com

# Build
docker build -t fastapi-app:latest app/

# Tag with ECR repository
docker tag fastapi-app:latest \
  ACCOUNT_ID.dkr.ecr.us-east-1.amazonaws.com/fastapi-app:$(git rev-parse --short HEAD)

# Push
docker push ACCOUNT_ID.dkr.ecr.us-east-1.amazonaws.com/fastapi-app:$(git rev-parse --short HEAD)
```
